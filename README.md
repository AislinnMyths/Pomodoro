# Pomodoro Timer 

A simple Pomodoro timer to help improve focus and productivity. I built this project as part of my self-learning journey in web development.

## ✨ Features (work in progress)
- Start, pause, and reset the timer  
- work sessions + breaks  
- Simple and responsive design
- Progress display and sound notification 


## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  

## 🔮 Future Improvements
- Add custom session lengths
- Add task tracking or statistics
- Barra de progreso animada
- Contador de ciclos completados
